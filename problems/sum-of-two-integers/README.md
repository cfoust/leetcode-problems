# Sum of Two Integers
[Leetcode page](https://leetcode.com/problems/sum-of-two-integers/description)

Calculate the sum of two integers _a_ and _b_ , but you are **not allowed** to
use the operator `+` and `-`.

**Example:**  
Given _a_ = 1 and _b_ = 2, return 3.
