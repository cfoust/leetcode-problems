# Sort List
[Leetcode page](https://leetcode.com/problems/sort-list/description)

Sort a linked list in _O_ ( _n_ log _n_ ) time using constant space
complexity.

**Example 1:**

    
    
    **Input:** 4- >2->1->3
    **Output:** 1- >2->3->4
    

**Example 2:**

    
    
    **Input:** -1- >5->3->4->0
    **Output:** -1- >0->3->4->5
